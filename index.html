Publications

Model-agnostic Characterization of Fairness Trade-offs
Joon Sik Kim, Jiahao Chen, Ameet Talwalkar
Preprint, 2020.

Proposed a model-agnostic perspective of analyzing different types of trade-offs in ML fairness, which yields additional theoretical results in impossibilities among fairness definitions.

Efficient Topological Layer based on Persistent Landscape
Kwangho Kim, Jisu Kim, Joon Sik Kim, Frederic Chazal, Larry Wasserman
Preprint, 2020.

Proposed a novel topological layer for deep neural networks based on persistent landscapes, which is able to efficiently capture underlying toplogical feautres of the input data, with stronger stability guarantees.

Diagnostic Curves for Black Box Models
David Inouye, Leqi Liu, Joon Sik Kim, Bryon Aragam, Pradeep Ravikumar
Neural Information Processing Systems (NeurIPS) Workshop on Safety and Robustness in Decision Making, 2019.

Proposed a framework for finding a univariate diagnostic curve in the input space that maximally violates certain functional properties of a black box model by defining multiple utility measures.

Representer Point Selection for Explaining Deep Neural Networks
Chih-kuan Yeh*, Joon Sik Kim*, Ian E.H. Yen, Pradeep Ravikumar
Neural Information Processing Systems (NeurIPS), 2018
published ver. / blog post / poster / code

Proposed a method that decomposes the pre-activation prediction of a neural network into a linear combination of activations of training points, with the weights corresponding to what we call representer values for intuitive interpreation of deep neural network prediction.

A Rotation Invariant Latent Factor Model for Moveme Discovery from Static Poses
Matteo Ruggero Ronchi, Joon Sik Kim, Yisong Yue
IEEE International Conference on Data Mining (ICDM), 2016
published ver. / project page / ppt / poster / code / dataset

Proposed a method to discover a set of rotation-invariant 3-D basis poses that can characterize the manifold of primitive human motions, or movemes, from a training set of 2-D projected poses obtained from still images taken at various camera angles.
